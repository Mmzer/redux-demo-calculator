# redux-demo-calculator
一个使用react和redux实现的仿iphone6s计算器

## Features
- React base es6
- redux
- react-redux
- webpack
- hot reloader
- babel
- scss
- autoprefixer
- file to base64


## Config

	webpack.base.config.js


## Clone

```bash
$ git clone https://github.com/Mmzer/redux-demo-calculator.git
$ cd redux-demo-calculator
$ npm install
```

## Start

```bash
$ npm start
```

## react-hot-reloader

```bash
$ npm run dev
```

## Build
```bash
$ npm run build
```
